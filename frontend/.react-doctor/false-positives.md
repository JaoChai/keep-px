# React Doctor — False Positives

Rules + code shapes that have been verified safe in this codebase. When react-doctor flags one of these, skip it.

## react-doctor/query-mutation-missing-invalidation

**Skip when** the mutation's `onSuccess` does any of the following — there's no TanStack cache that goes stale:

- Redirects via `window.location.href = ...` (page unmounts before stale data matters). Example: `src/hooks/use-billing.ts` checkout/portal/slots mutations route to Stripe.
- Writes to a Zustand store via `useAuthStore.getState().setCustomer(...)` — auth state lives outside TanStack. Examples: `src/hooks/use-auth.ts` `useRegenerateAPIKey`, `src/pages/auth/LoginPage.tsx` `devLogin`.
- Returns a one-shot value the caller uses directly without caching:
  - `src/hooks/use-upload.ts` `useUploadImage` / `useUploadImages` — returns URL string, caller persists via separate form submit.
  - `src/hooks/use-replays.ts` `useReplayPreview` — preview-only, no list to invalidate.
  - `src/hooks/use-pixels.ts` `useTestPixel` — test event roundtrip, no DB write.

## react-doctor/exhaustive-deps

**Skip when** the file already has `// eslint-disable-next-line react-hooks/exhaustive-deps` directly above the hook AND one of these patterns applies:

- **Mount-only restore-draft `useEffect(..., [])`** — intentional one-shot localStorage hydration. Examples: `src/pages/BlockEditorPage.tsx:120`, `src/pages/SalePageEditorPage.tsx:196`.
- **`useMemo` with deliberately wider deps than read** — used to recompute memo on flag changes (e.g. `isDirty`, `hasUserEdited`) without including them in the body. Example: `src/pages/SalePageEditorPage.tsx:127` `draftData` memo.

## react-doctor/no-react19-deprecated-apis (forwardRef)

**Skip in shadcn/ui generated components** under `src/components/ui/*` (radix wrappers like `select.tsx`, `sheet.tsx`, `popover.tsx`, `tooltip.tsx`, `dialog.tsx`, etc.). These are generated by the shadcn CLI and use `React.forwardRef` to forward refs to Radix primitives. Will be auto-updated when shadcn ships React 19-native versions. Hand-rolled components (e.g. `button.tsx`) have been migrated.

## react-doctor/only-export-components (router files)

**Skip in `src/router.tsx`** — by design the file exports `router` (a non-component config object) and assigns `lazy(...)` to const bindings. Fast Refresh limitation does not apply to routing config files.

## deslop/unused-dev-dependency

**Skip `react-doctor` itself** in `package.json` — it's invoked via `npx react-doctor@latest` in `npm run doctor`, not imported as a module.
